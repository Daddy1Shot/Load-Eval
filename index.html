Settings & Baseline

Set once. Tune anytime. All values editable.

Vehicle
Cargo weight only slows the loaded leg. At max load the van loses this many MPG (ProMaster 1500: ~3, i.e. 13→10). Empty deadhead legs always use Empty MPG. Tune after a few heavy runs.
Auto-added to the calculated drive time, since the API only knows driving — not the wait at the Lowe's desk. Lowe's runs slow, so 30 is a fair default. Still editable per load.
Home Base
Relocating soon? Update this one field and every new load uses it.
Saved Pickups (Lowe's, etc.)
Pickups are usually a known Lowe's. Save the ones you get dispatched from, then tap to fill the Pickup field at offer time — even before you accept and the app reveals it.
Decision Thresholds ($ / total mile)
A load can pass $/mile but still fail here if it eats too much time.
Fuel
